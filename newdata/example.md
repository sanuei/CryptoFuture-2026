# 示例文档

tags: 示例, 测试

这是一个示例 Markdown 文档。你可以将类似的文档拖拽到网页的上传区域来查看。

## 功能说明

当你上传 Markdown 文件时，系统会：

1. **自动提取标题**：从文件的第一行 `# 标题` 中提取
2. **提取日期**：从文件名或内容中提取日期
3. **提取标签**：从 `tags:` 行中提取标签
4. **生成摘要**：自动生成文档摘要

## 使用技巧

- 在文件开头使用 `# 主标题` 格式
- 使用 `tags: 标签1, 标签2` 来设置标签
- 文件名可以包含日期，如：`2024-05-20-文档名.md`
