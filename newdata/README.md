# NewData 文件夹

这个文件夹用于存放 Markdown 文档。

## 使用方法

1. **在网页中上传**：
   - 打开网页首页
   - 找到"上传 Markdown 文档"区域
   - 将 `.md` 文件拖拽到该区域，或点击选择文件
   - 文件会自动解析并添加到文档列表中

2. **文件格式要求**：
   - 文件扩展名必须是 `.md`
   - 建议在文件开头使用 `# 标题` 格式
   - 可以在文件中添加 `tags: 标签1, 标签2` 来设置标签
   - 可以在文件名中包含日期，如：`2024-05-20-文档标题.md`

## 示例 Markdown 文件

```markdown
# 文档标题

tags: 标签1, 标签2

这是文档的内容...

## 章节标题

更多内容...
```

## 注意事项

- 文件会保存在浏览器的 localStorage 中
- 上传的文件会立即显示在文档列表中
- 支持同时上传多个文件
